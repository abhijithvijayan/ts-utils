## Installation

Ensure you have [Node.js](https://nodejs.org) 10 or later installed. Then run the following:

```
# via npm
npm install node-module-ts-starter

# or yarn
yarn add node-module-ts-starter
```

## Usage

```js
const unicornFun = require('node-module-ts-starter');

unicornFun('unicorns');
//=> 'unicorns & rainbows'
```
